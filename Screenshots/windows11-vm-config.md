Final hardware configuration for the Windows 11 practice VM (VMID 100) before first boot — q35 machine type, OVMF (UEFI) BIOS, VirtIO SCSI disk (64GB), VirtIO NIC, 4096MB RAM. TPM and EFI disk entries (set earlier in the process) are just above this section of the config.
