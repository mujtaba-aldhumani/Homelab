## Lessons Learned

- GitHub Desktop does not replace Git.
- Obsidian Git requires Git to be installed and available in PATH.
- Changes must be pushed from one device and pulled on another.
- Configuration files are common sources of merge conflicts.
- Documentation should be established before infrastructure deployment.
